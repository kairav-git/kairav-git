I'm a **Frontend Developer** based in India with a passion for turning ideas into clean, performant, and visually engaging web experiences. I specialize in building **scalable Angular applications** and crafting component-driven UIs with a strong eye for design.

- 🔭 &nbsp;Currently building interactive **3D portfolio experiences** with Babylon.js
- 🌱 &nbsp;Deepening skills in **Python**, **FastAPI**, and AI-driven backends
